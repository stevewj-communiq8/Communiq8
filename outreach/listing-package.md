## 4. Transfer Process Notes

### For .com and .net (Verisign Registry)
```
1. Unlock domain at IONOS SE control panel
2. Disable transfer lock (remove clientTransferProhibited)
3. Generate EPP auth code from IONOS SE
4. Provide auth code to buyer (via escrow/platform)
5. Buyer initiates transfer at their registrar
6. Approve transfer request at IONOS SE
7. Transfer completes in 5-7 days
```

### For .biz (Afilias/GoDaddy Registry)
```
Same process as .com/.net:
1. Unlock domain at IONOS SE
2. Obtain EPP auth code
3. Provide to buyer via escrow
4. Buyer initiates transfer
5. Complete in 5-7 days
```

### For .co.uk (Nominet UK)
```
Different process — no EPP code needed:
1. Buyer provides their nominated registrar's IPS tag
2. Seller changes IPS tag at IONOS SE to buyer's registrar
3. Transfer completes within 24 hours (typically)
4. No escrow delay — fast and simple
```

### Escrow Recommendations
| Platform | Escrow Fee | Notes |
