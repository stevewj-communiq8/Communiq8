## 7. Transfer Instructions

### .com and .net (Verisign)
1. Unlock domain at IONOS SE (remove clientTransferProhibited)
2. Obtain EPP auth code from registrar
3. Buyer initiates transfer at their registrar
4. Seller confirms transfer request
5. Transfer completes within 5-7 days

### .biz (Afilias)
1. Unlock domain at IONOS SE
2. Obtain EPP auth code
3. Standard transfer process via any ICANN-accredited registrar
4. Transfer completes within 5-7 days

### .co.uk (Nominet UK)
1. Request IPS tag change at IONOS SE to buyer's nominated registrar
2. No EPP code required — Nominet uses IPS tag system
3. Transfer completes within 24 hours typically

### Escrow Process
- Recommended: **Escrow.com** (1.5-3% fee)
- Or use platform escrow: Sedo (included), Afternic (included), DAN.com (included)
- Fees split or borne by buyer depending on negotiation
