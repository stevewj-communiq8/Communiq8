# Communiq8 — Visual Asset Briefs

## Brand Visual Identity

| Element | Specification |
|---|---|
| **Primary Colour** | Deep Navy (#0F172A) |
| **Accent Colour** | Gold (#F59E0B / #D97706) |
| **Secondary** | Cool Gray (#64748B) |

## Asset 1: Hero Brand Card (Static)

**Use:** Profile picture, cover image, post templates
**Format:** 1080×1080px square
**Description:**
Dark navy background with subtle gradient. In the centre, "Communiq8" in Playfair Display Bold — white with "8" in gold. Below in smaller Inter text: ".com  .co.uk  .net  .biz". Subtle geometric connection lines (communication nodes) in cool gray radiating from centre. Gold thin border accent.

---

## Asset 2: Domain Cards Carousel Template

**Use:** Multi-slide carousel posts
**Format:** Each slide 1080×1080px
**Description:**
4-slide carousel where each slide features one domain prominently. Each card: dark navy with a subtle circular accent. Domain name displayed large: "communiq8.com" with the TLD in gold. Below: 2-bullet value point. Bottom: small flag emoji corresponding to TLD (🌐 for .com, 🇬🇧 for .co.uk, 🔗 for .net, 💼 for .biz).
