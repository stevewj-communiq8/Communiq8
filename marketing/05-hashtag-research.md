# Communiq8 — Hashtag Research

## Strategy Overview

We use a **layered hashtag strategy** for each post:
- **3-5 broad hashtags** (high volume, high reach)
- **3-5 niche hashtags** (medium volume, high engagement)
- **2-3 branded hashtags** (zero competition, owned space)
- **Total per post: 8-15 hashtags** (Instagram best practice)

## Hashtag Rules & Best Practices

1. **Never exceed 15 hashtags** per Instagram post (algorithm penalty risk)
2. **Place hashtags in the first comment** for cleaner captions (optional — test both)
3. **Rotate hashtag sets** — don't use the same 15 every time
4. **Include 1-2 local Kuwait tags** on every post for location relevance
5. **Track performance** — identify which sets drive profile visits
6. **TikTok uses fewer tags** (3-5 max): focus on #Communiq8 #PremiumDomain #Kuwait #Q8 #Domains
7. **X/Twitter uses minimal-to-no hashtags** — 1-2 max per tweet. Focus on high-value tags like #Domains #Kuwait #StartupLife. Threads should avoid hashtags in the body.
8. **Avoid banned or spammy tags** (e.g., #follow4follow, #domainsforsale can be low quality)
9. **Always tag @communiq8domain** in all post copy where the handle is mentioned
