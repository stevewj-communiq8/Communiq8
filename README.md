# Communiq8 Domain Portfolio

A premium bundle of four aged, keyword-rich domains (communiq8.com, .co.uk, .net, .biz) spanning the top TLDs.

## Landing Page
The landing page is a static site designed for high conversion and premium feel.

### Deployment
This repository is prepared for deployment on **Netlify**.
- **Build Settings**: The site can be served from the root directory.
- **Forms**: Uses [Netlify Forms](https://docs.netlify.com/forms/setup/) for contact form submissions.

## Repository Structure
- `/`: Landing page source files (HTML, CSS, JS).
- `/marketing/`: Content strategy, social media calendar, and visual briefs.
- `/outreach/`: Marketplace research and email outreach templates.
- `server.js`: Node.js server for local development and form submission testing.

## Local Development
1. Install dependencies: `npm install`
2. Run the server: `node server.js`
3. Access the site at `http://localhost:3000`
